# Azure VM Provisioning Project

## Project Overview
This project demonstrates the provisioning and management of an Ubuntu Virtual Machine in Microsoft Azure. The virtual machine was configured with SSH access, monitored using Azure monitoring tools, and used to deploy an Apache web server.

## Objectives
- Create an Ubuntu Virtual Machine in Microsoft Azure
- Configure networking and SSH access
- Perform Linux administration tasks
- Install and manage Apache Web Server
- Monitor VM performance using Azure Monitor
- Explore Azure Auto-Scaling concepts

## Technologies Used
- Microsoft Azure
- Ubuntu Server 24.04 LTS
- SSH
- Apache Web Server
- Azure Monitor
